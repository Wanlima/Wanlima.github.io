# 1. nil、Nil、Null、NSNull 的区别？

标志 值 含义
NULL (void *)0 C 空指针
nil (id)0 Objective-C 对象为空
Nil (Class)0 Objective-C 类为空
NSNull [NSNull null] 用来表示空值的 Objective-C 对象

WARNING

Objective-C 中不存在 NIL 或 NSNil 这两个符号！！！

详情见 Objective-C 中 NULL、nil、Nil、NSNull 的定义及不同 (opens new window)。

# 2. atomic 修饰的属性是绝对安全的吗？为什么？

atomic 在调用属性的 getter 和 setter 方法时，会加上同步锁，保证同一时刻只能有一个线程调用属性的读/写方法。保证了读和写的过程是可靠的。但并不能保证数据一定是可靠的。

注意

[array objectAtIndex: index] 不是线程安全的。
